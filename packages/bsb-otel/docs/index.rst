BSB OpenTelemetry integration documentation
============================================

This website hosts the documentation references related to the ``bsb-otel`` package.

``bsb-otel`` is a plugin of the ``BSB``.
It provides OpenTelemetry integration utilities for BSB, including trace export and replay tools.
