# bsb-otel

`bsb-otel` is a plugin of the [BSB](https://github.com/dbbs-lab/bsb).
It provides OpenTelemetry integration utilities for BSB, including trace export and replay tools.

Comprehensive documentation is available at:

* [BSB Documentation](https://bsb.readthedocs.io/en/latest)
* [bsb-otel Documentation](https://bsb-otel.readthedocs.io/en/latest)
